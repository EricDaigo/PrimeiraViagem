# Primeira Viagem

Site de vendas do guia **"50 Dicas Que Ninguém Te Contou Sobre Maternidade"**, desenvolvido para mães de primeira viagem.

## Páginas

| Arquivo | Descrição |
|---|---|
| `site/index.html` | Landing page principal de vendas |
| `site/afiliados.html` | Kit do afiliado com materiais de divulgação |
| `site/obrigado.html` | Página de confirmação pós-compra |
| `site/termos.html` | Termos de uso e política de privacidade |

## Como visualizar

Abra qualquer arquivo `.html` diretamente no navegador — o site não depende de servidor ou build.

## Tecnologias

- HTML5
- CSS3 (inline, sem dependências externas)
- JavaScript vanilla
- Google Fonts (League Spartan)
